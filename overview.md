## 核心功能

### 需求一：用户注册登录
- H5端注册（账户名唯一+密码），BCrypt加密存储
- 登录需 账户名+密码+滑块图片验证（纯Java AWT生成，容差±5px）
- 注册成功自动登录返回JWT（role=MEMBER）
- 临时用户与注册会员共存，未登录仍可点餐

### 需求二：余额点卡支付
- **后台批量创建点卡**：指定额度+张数，自动生成卡号（RC+日期+随机数）+卡密（16位随机）
- **后台发放点卡**：可指定发放给某个注册会员
- **H5兑换点卡**：用户输入卡号+卡密，校验通过余额入账
- **H5余额支付**：余额充足时扣款+订单状态变更（同一事务），余额不足提示
- **余额变动记录**：充值/消费明细

## 用户下一步建议

1. **H5注册登录**：访问 http://localhost:5174 → 「我的」页面 → 点击注册 → 注册成功后登录
2. **后台点卡管理**：登录 http://localhost:5173（admin/admin123）→ 左侧「点卡管理」→ 批量创建点卡 → 发放给会员
3. **H5兑换点卡**：会员登录后 → 「我的」→ 兑换点卡 → 输入卡号卡密 → 余额入账
4. **H5余额支付**：下单后 → 支付页选择「余额支付」→ 余额扣款成功
5. **余额记录**：「我的」→ 余额记录 → 查看充值/消费明细
